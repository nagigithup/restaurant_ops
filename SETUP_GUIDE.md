# Restaurant Operations Setup Guide

1. Create or select an ERPNext Company.
2. Create warehouses for each branch: Raw Materials, WIP, and Finished Meals.
3. Create meal Items and raw material Items.
4. Create BOM recipes for meal Items.
5. Create a Restaurant Branch and set its default warehouses, customer, supplier, and cost center.
6. Open Restaurant Settings and set automation/account defaults.
7. Staff should work from the Restaurant Operations workspace.

Recommended daily flow: Daily Purchases, Daily Kitchen Production, Daily Sales Summary, then Kitchen Wastage.
